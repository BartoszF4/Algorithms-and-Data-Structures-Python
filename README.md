# Algorithms and Data Structures in Python

This repository contains university lab assignments, algorithm implementations, and data structure exercises developed during the Algorithms and Data Structures (ADS) course. The goal is to demonstrate foundational computer science principles, clean code practices, and efficient problem-solving.

## Repository Structure

The project is organized by laboratory sessions, with each folder focusing on specific algorithmic paradigms or data structures:

### 📁 [Lab 01](./Lab-01) - Introductory Algorithmic Problems
* **Task 1 (`Lab1Task1.py`)**: Implementation and analysis of introductory problem-solving logic.
* **Task 2 (`Lab1Task2.py`)**: Algorithmic exercise focusing on foundational programming constructs and efficiency.

---

## Technical Highlights
* **Language**: Python 3.x
* **Core Focus**: Time and space complexity optimization, proper utilization of built-in data types, and implementing robust error handling for algorithmic edge cases.
